# Instrukcja wrzucenia strony na GitHub (Hosting za darmo)

Postępuj zgodnie z tymi krokami, aby Twoja strona była dostępna dla każdego pod linkiem `twojanazwa.github.io`.

### Krok 1: Przygotuj pliki
Upewnij się, że w folderze `E:\do antigrafity\assets` masz:
- `avatar.png` (zdjęcie profilowe)
- `song.mp3` (piosenka Akiaura - Sleepwalker)

### Krok 2: Załóż repozytorium na GitHub
1. Wejdź na [github.com](https://github.com/) i zaloguj się.
2. Kliknij przycisk **"New"** (zielony) obok "Repositories".
3. W polu **Repository name** wpisz np. `moj-profil` lub `stretchydev`.
4. Upewnij się, że zaznaczone jest **Public**.
5. Kliknij **"Create repository"**.

### Krok 3: Wgraj pliki
1. Na nowej stronie zobaczysz napis "Get started by uploading an existing file". Kliknij niebieski napis **"uploading an existing file"**.
2. Przeciągnij **wszystkie pliki i foldery** z `E:\do antigrafity` (czyli `index.html`, folder `css`, folder `js`, folder `assets`) do okna przeglądarki.
3. Poczekaj, aż wszystko się załaduje.
4. Na dole w polu "Commit changes" wpisz `First commit` i kliknij zielony przycisk **"Commit changes"**.

### Krok 4: Włącz stronę (GitHub Pages)
1. W Twoim repozytorium kliknij zakładkę **"Settings"** (na górze).
2. Z menu po lewej stronie wybierz **"Pages"**.
3. W sekcji "Build and deployment" upewnij się, że pod "Source" jest wybrany **"Deploy from a branch"**.
4. Pod "Branch" wybierz `main` (lub `master`) oraz `/ (root)`.
5. Kliknij **"Save"**.

### Gotowe!
Poczekaj około 1-2 minuty. Na górze strony w zakładce "Pages" pojawi się link (np. `https://twojanazwa.github.io/moj-profil/`). **To Twój link do strony!**

---
**Tip:** Jeśli kupisz domenę za 5 zł, możesz ją podpiąć w tej samej zakładce "Pages" w polu "Custom domain"!
